## Käyttöohje

Projektin buildaus:

`./gradlew build`

Projektin käynnistäminen komentoriviltä:

`./gradlew run`

Testien ajaminen:

`./gradlew test`

Checkstylen ajaminen:

`./gradlew checkstyleMain`

Jar generointi (kohteeseen build/libs/Vinkkikirjasto-all.jar):

`./gradlew shadowJar`
